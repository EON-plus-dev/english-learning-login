# LinguaFlow — Login Page

Glassmorphism login page for an English learning app. Pure HTML / CSS / JS, zero dependencies.

## Features

- Glass-effect card with backdrop blur
- Animated background blobs
- Smooth entrance animations (card, fields, button)
- Password visibility toggle
- Ripple effect on submit button
- Client-side validation with shake animation
- Google / Apple social login buttons (UI only)
- Success state after login
- Mock credentials panel with auto-fill
